# 像宿
像宿：在线动漫弹幕APP。多番剧源，多弹幕，高清无广告，还有海量精选热门COSPLAY、插画。追番看番看图必备软件。

## 支持平台

- Android
- Windows
- macOS (未测试)
- iOS (未测试，需要自签名)
- Linux (未测试)

## 功能

- [x] 番剧索引
- [x] 番剧搜索
- [x] 分集播放
- [x] 倍速播放
- [x] 观看记录
- [x] 追番管理
- [x] 番剧弹幕
- [x] 弹幕过滤
- [x] 分集评论
- [x] 图片收藏
- [x] 离线缓存
- [ ] 番剧评分
- [ ] ...
